Access keys act like:
Access Key ID ~ Username 
Secret Access Key ~ Password
for when a root/non-root user want to access the AWS through CLI or use AWS SDK.

- Access key is generated through the AWS Console
- Users manage/generate their own access keys

Steps to create access keys:

1. users->user->security cred->create access keys
2. copy creds
3. `aws configure` - in the CLI
