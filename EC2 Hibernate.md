**EBS (Elastic Block Store)** is a storage service in AWS used with EC2 instances.
### Simple idea
Think of EBS as a **virtual hard disk** that you attach to an EC2 instance.


**EC2 Hibernate** - In short, a copy of the RAM/Memory is dumped to EBS when the instance is stopped. And that RAM copy is copied from EBS when booting up the instance. Almost as if the Instance was never stopped.
