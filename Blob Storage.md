# 3 vs Blob Storage

## 1. 📦 Core Concept (Very Similar)

- **S3 (Simple Storage Service)** → Stores data as **objects in buckets**
- **Blob Storage** → Stores data as **blobs in containers**

👉 Mapping:

- Bucket (S3) ≈ Container (Blob)
- Object (S3) ≈ Blob (Azure)
